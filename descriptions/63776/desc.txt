// Handles object size validation and unknown data extraction for different DWG versions.
// PRE (R_11) handles older DWG versions without CRC checks, verifying the object size
// and extracting any unknown trailing data if the size is incorrect.
// LATER_VERSIONS handles newer versions, checking for valid sizes and performing CRC validation.
// If the actual size is larger than expected, extracts the unknown trailing data into `obj->unknown_rest`.
